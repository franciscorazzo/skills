# Skills

Coleção de Agent Skills de Francisco Razzo.

## Skills disponíveis

- **[debate-logic-analyzer](debate-logic-analyzer/)** *(pt-br)* — Auditoria lógica e epistemológica de textos argumentativos: diagrama de Nolt, formalização proposicional e de predicados em Unicode, e teste do Mito do Dado (Sellars). Fecha com veredicto integrado.
- **[debate-logic-analyzer-en](debate-logic-analyzer-en/)** *(English)* — English version of the logic and epistemology audit: Nolt diagram, propositional/predicate formalization in Unicode, and the Myth of the Given test (Sellars). Closes with an integrated verdict.
- **[concept-auditor](concept-auditor/)** *(pt-br)* — Auditoria conceitual: mapa dos conceitos-chave, teste de estabilidade (equívoco, deslizamento semântico) e teste de definição (definição persuasiva, conceito essencialmente contestado). Verifica se os termos que carregam o argumento se mantêm estáveis.

## Adicionar uma skill

Veja [CONTRIBUTING.md](CONTRIBUTING.md). Em resumo: copie `_template/`, escreva o `SKILL.md`, rode `pwsh scripts/package-skills.ps1` para validar e empacotar, e registre a skill aqui.
